운동 기록하기 (오늘 탭)

세트·무게·횟수를 탭탭탭으로

운동 시작 → 기록

  1. ＋ 운동 시작하기 누르기
  2. 운동 선택 창의 200개 운동을 부위와 기구(맨몸·덤벨·바벨·밴드·머신 등)로 좁히거나 이름으로 검색해 고르기. 없으면 새 운동 만들기로 직접 추가
  3. 자주 하는 운동은 운동 기록 화면의 ★을 눌러 즐겨찾기. 다음에는 운동 선택 창의 ⭐ 즐겨찾기나 🕘 최근 운동에서 바로 찾을 수 있어요.
  4. 세트의 RPE 기록을 누르면 RPE 6~10과 RIR이 함께 저장돼요. 이전 기록이 충분하면 다음 권장 무게도 자동으로 보여줘요.
  5. 루틴을 만들 때 운동마다 목표 세트·횟수·휴식시간을 저장하면 시작할 때 자동으로 준비돼요.
  6. 운동이 추가되면 세트별로 무게·횟수 입력 — ＋ / － 버튼으로 빠르게 조절(무게 ±2.5, 횟수 ±1)
  7. 세트를 마치면 체크(완료) 탭 → 휴식 타이머가 자동으로 시작
  8. 운동이 끝나면 위쪽 완료 버튼 → 요약 확인

알아두면 좋은 것

  • 세트 종류: 세트 번호를 탭하면 일반 → W 워밍업 → D 드롭 → F 실패 순서로 바뀌어요. 워밍업은 볼륨·기록에서 제외돼요.
  • 지난 기록 자동 표시: 같은 운동을 하면 지난번 무게·횟수가 옆에 보여서 참고하기 좋아요.
  • 개인기록(PR) 🏆: 역대 최고 무게나 1RM을 넘기면 축하 배너가 떠요.
  • 휴식 타이머: －15초 건너뛰기 ＋15초로 조절. 기본 시간은 설정에서 바꿔요.
그냥 운동한 시간만 남기고 싶다면? 세트 기록 없이 🏅 챌린지에서 요일 칸에 시간만 톡 쳐도 인증돼요.
🍱

식단 기록하기 (식단 탭)

먹은 것 + 칼로리·탄단지

음식 추가하는 2가지 방법

  1. 직접 추가 — 끼니(아침·점심·저녁·간식) 아래 ＋ 직접 추가 → 음식 이름·칼로리·탄단지 입력
  2. 📷 사진으로 칼로리 (AI) — 📷 사진으로 칼로리 → 새로 촬영하거나 앨범의 기존 사진을 골라도 돼요 → AI가 이름·칼로리·영양을 자동으로 채워줘요(값은 확인·수정 가능)

목표와 요약

  • 맨 위 칼로리 링과 탄수·단백·지방 막대로 오늘 먹은 양을 한눈에 봐요.
  • 오른쪽 위 🎯 목표에서 하루 목표 칼로리·영양을 정해요.
  • 오른쪽 위 📸 저장을 누르면 그날 식단만 예쁜 카드 이미지로 만들어 사진 앱에 저장하거나 카톡으로 공유할 수 있어요.
  • ‹ ›로 지난 날짜의 식단도 보고 기록할 수 있어요.
근육 키우려면 보통 몸무게 1kg당 단백질 1.6~2.2g을 권장해요. (예: 70kg → 112~154g)

통계 · 1RM · 디로딩 (통계 탭)

내 성장과 몸 상태 보기

🧮 1RM 계산기

통계 탭 위쪽 🧮 1RM(또는 오늘 탭 버튼) → 무게 + 반복 횟수 입력하면 추정 1RM과 함께 %별 목표 무게 표(예: 80% = 몇 kg × 8회)가 나와요. 무게 세팅할 때 딱이에요.

🔴 피로도 · 디로딩 신호

통계 탭 위쪽 카드에서 오늘 컨디션을 1~5로 눌러두면, 연속 훈련 주수·볼륨 변화·피로도를 종합해 신호를 줘요:

  • 🟢 양호 — 지금 페이스 유지, 조금씩 늘려도 OK
  • 🟡 주의 — 피로 쌓이는 중, 한 주 지켜보기
  • 🔴 디로딩 추천 — 이번 주는 볼륨 40~50% 줄이고 휴식·영양 늘리기
디로딩이란? 일부러 강도를 낮춰 쉬어가는 주간. 과훈련·부상을 막고 더 강하게 복귀하는 검증된 방법이에요.

그 외 통계

  • 운동 잔디 — 최근 1년 운동한 날을 색으로 표시
  • 근육 히트맵 — 이번 주 어느 부위를 많이 자극했는지
  • 개인기록(PR)과 운동별 성장 그래프(추정 1RM 추이)
  • 위쪽 📅 기록으로 날짜별 달력·인증 사진도 볼 수 있어요.
🏅

그룹 챌린지 (챌린지 탭)

운동방 친구들과 순위 겨루기

매일 인증하기

  1. 먼저 닉네임 설정(안 했다면 안내가 떠요)
  2. 요일 칸 중 오늘 날짜 칸을 탭
  3. 오늘 운동 시간 입력(30분·1시간 버튼 또는 직접) — 원하면 📷 사진도 첨부
  4. 인증하기 → 출석부·순위에 바로 반영!
사진을 올리면 순위 +1점 🔥 꾸준함이 순위로 보여요.

👑 운동방을 만드는 사람 (방장)

  1. 챌린지 탭 아래 ＋ 챌린지 만들기
  2. 생기는 6자리 코드를 오픈채팅방에 공유
  3. 친구들이 그 코드로 참여하면 같은 순위표에 모여요

🙋 참여하는 사람 (멤버)

  1. 방장에게 받은 6자리 코드를 참여 칸에 입력
  2. 매일 위처럼 인증하면 끝
  3. 🏆 리더보드(순위)와 📋 출석부(전체 표)로 서로 확인

📷 인증 이미지 공유

챌린지 탭 오른쪽 위 📷 이미지 공유를 누르면 이번 주 내 운동 인증을 예쁜 카드 이미지로 만들어 오픈채팅·SNS에 바로 공유할 수 있어요.

실시간 그룹 순위는 인터넷이 연결돼 있을 때 동기화돼요. 안 보이면 새로고침을 눌러보세요.

📷 음식 칼로리 (식단 탭)

  • 촬영하거나 앨범(갤러리)의 기존 사진을 골라도 돼요
  • 음식이 화면에 꽉 차게, 위에서 밝은 곳에서 찍은 사진이 정확해요
  • 한 접시에 여러 음식이면 합쳐서 추정해요
  • 직접 추가할 때도 🖼️ 사진 첨부로 기존 이미지를 붙일 수 있어요
  • AI 값은 추정치예요 — 나온 숫자를 보고 직접 수정하면 더 정확

🤖 자세 체크 (오늘/홈)

  • 측면(옆)에서 전신이 다 보이게 촬영
  • 동작의 가장 낮은 지점(예: 스쿼트 앉은 자세)에서 찍기
  • 운동 종류를 고르고 사진을 올리면 점수 · 잘한 점 · 개선점 · 실행 큐를 줘요
AI 피드백은 참고용이에요. 통증·부상 위험이 있으면 반드시 전문가(트레이너·의료진)에게 확인하세요.
인터넷이 없거나 일시적 문제로 AI가 안 되면, 앱이 자동으로 직접 입력으로 바꿔주니 기록은 계속할 수 있어요.

설정 & 백업 (설정 탭)

내 취향대로 + 안전하게

  • 내 프로필 — 닉네임·지역·나이·성별
  • 주간 목표 — 한 주에 몇 번 하면 "달성"인지 (주 1~7회)
  • 기본 휴식 시간, 무게 단위(kg/lb)
  • 기기 간 동기화 — 코드로 다른 기기와 내 기록을 이어서 보기
  • 백업 — 기록 내보내기(.json) / 불러오기. 폰 바꾸기 전에 한 번!

자주 묻는 질문

궁금한 걸 눌러보세요

회원가입 해야 하나요?

아니요. 로그인·가입 없이 바로 씁니다. 기록은 내 폰(브라우저)에 자동 저장돼요.

기록이 사라지지 않나요?

같은 브라우저로 열면 계속 남아있어요. 다만 브라우저 데이터를 지우면 사라질 수 있으니, 중요하면 설정 → 백업(내보내기)를 해두세요. 그룹 챌린지에 참여하면 그룹 기록은 클라우드에도 동기화됩니다.

폰을 바꾸면 기록을 옮길 수 있나요?

네. 설정 → 기기 간 동기화 코드를 만들어 새 폰에서 그 코드로 이으면 돼요. 또는 백업 내보내기 → 새 폰에서 불러오기도 가능합니다.

인터넷이 없어도 되나요?

내 운동·식단 기록은 오프라인에서도 돼요. 그룹 순위·AI 분석·사진 업로드만 인터넷이 필요합니다.

돈이 드나요?

앱은 완전 무료예요. AI(자세·칼로리)는 관리자가 준비해둔 기능으로, 사용자는 추가 비용 없이 그냥 쓰면 됩니다.

운동만 하고 세트 기록은 귀찮아요.

괜찮아요! 챌린지 탭에서 요일 칸에 운동한 시간만 톡 쳐도 인증돼요. 세트·무게 상세 기록은 원할 때만 하면 됩니다.

사진은 누구나 볼 수 있나요?

같은 그룹 챌린지 코드에 참여한 사람끼리 인증 사진을 볼 수 있어요. 코드를 모르는 사람은 볼 수 없습니다.
